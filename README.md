## 核心功能

### 1. 号码频次分析
- 输入号码或生肖，实时统计 1-49 各号码出现频次
- 支持混合输入：数字 + 生肖（空格分隔）
- 上限 5000 个号码，超限自动截断并提示
- **有效数字计数（不去重）** + **分析结果自动去重**

### 2. 多维度筛选（斩杀系统）
| 筛选维度 | 选项 |
|---------|------|
| 杀码 | 自定义斩杀号码 |
| 生肖 | 鼠、牛、虎、兔、龙、蛇、马、羊、猴、鸡、狗、猪 |
| 头数 | 0-4 头单 / 头双 |
| 尾数 | 0-9 尾 |
| 数段 | 1-7 段 |
| 波色 | 红波单/双、蓝波单/双、绿波单/双 |
| 五行 | 金、木、水、火、土 |
| 半单双 | 合数单/双、大单/双、小单/双 |
| 合数 | 1-13 合 |

- 筛选条件实时叠加，命中次数显示角标
- 平均频次斩杀线自动分割高低频号码

### 3. 0 次统计
- 当不重复数字少于 49 个时，未出现号码自动列出
- 方便补全遗漏分析

### 4. 独苗守护特效
- 当仅有一个号码未被任何筛选命中时，触发飞入动画
- 金色粒子轨迹 + 脉冲高亮 + 震动着陆

### 5. 实时开奖
- 对接 `macaumarksix.com` API，获取最新开奖结果
- 蛋形 3D 号码球，带波色、生肖、五行标签
- **自动刷新**：每天 21:33:20 – 21:35:00 每 5 秒刷新，开完 7 码自动停止
- 离线缓存：断网时自动显示上次缓存数据

### 6. 开奖直播
- 内置 HLS / FLV 多源切换
- API 自动获取直播流地址
- 失败自动 fallback 到备选源

### 7. 历史开奖
- 按年份查询 2020 年至今全部开奖记录
- 分页展示，每页 15 期
- 波色 + 生肖 + 五行标签完整显示

### 8. 水泡粒子背景
- 五颜六色水泡粒子自底上升
- 自动适配性能，低端设备流畅运行

## 技术栈

| 技术 | 说明 |
|------|------|
| 纯原生 JS | 无框架依赖，单文件 HTML 即开即用 |
| Web Worker | 计算密集型分析卸载到独立线程 |
| Tailwind CSS | 原子化样式，响应式布局 |
| DocumentFragment | 批量 DOM 插入，减少重排 |
| Uint8Array / Uint16Array | 高性能数组计算 |
| localStorage | 状态持久化 + 开奖数据离线缓存 |
| AbortController | 请求超时控制 |

## 文件结构

```
.
├── index.html    # 入口页面
├── app.js        # 主线程逻辑（渲染、事件、UI）
├── data.js       # 静态数据（生肖、五行、波色、数段、号码属性）
├── style.css     # 主题样式（3D 球、动画、抽屉、响应式）
└── worker.js     # Web Worker（频次计算、筛选命中）
```

> 5 个文件全部放在同一目录下，无需构建工具，直接部署。

---

## 部署方式

### GitHub Pages（推荐）

1. 将 5 个文件上传到仓库根目录
2. Settings → Pages → Source 选择 `main` 分支，`/ (root)` 目录
3. 访问 `https://你的用户名.github.io/仓库名/`

### 本地运行

直接用浏览器打开 `index.html` 即可，无需服务器。

> ⚠️ 注意：部分浏览器限制本地 `file://` 协议的 Worker，建议用 Live Server 插件或 `python -m http.server`。
